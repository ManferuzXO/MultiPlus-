## рҹ“– DescripciГіn

**Multiplus** es una plataforma web de entretenimiento que simula un servicio de streaming. Permite navegar por catГЎlogos de **series**, **pelГӯculas** y **anime**, ver fichas de detalle de cada tГӯtulo y reproducir contenido directamente desde el navegador.

> рҹҡ« No requiere backend ni base de datos вҖ” es 100% frontend estГЎtico.

|---|---|
| рҹ“ұ **Responsive** | Adaptado para mГіvil, tablet y escritorio |
